How to Use This Page

This table displays a list of registered users who have signed up on the Media Matrix platform. The columns provide the following information:

  • Name: The user's full name.
  • Email: The user's email address.
  • Phone: The user's phone number.
  • Password: The password used by the user.
  • Confirm Password: Confirmation of the password.

The table is fully interactive and can be sorted and filtered based on your needs. Below is an example of the data you will see.
